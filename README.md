# Project Penunjang Kursus Android Level 1 di Alteration Studio
## Pertemuan 1 -  Pengenalan Android Studio

Kegiatan kursus Android Studio Beginner (Level 1) di [Alteration Studio](http://www.alterationstudio.web.id/) untuk pertemuan ke-1 berdurasi +-2 jam. Berikut adalah poin-poin yang disampaikan selama kursus :

- Pada pertemuan ini peserta kursus akan mengenal segala tools yang digunakan untuk membangung aplikasi Android. Praktikan juga akan diperkenalkan dengan workspace pada android studio.
- Tools yang perlu dipersiapkan pada pertemuan ini adalah JDK, Android Studio, dan Vysor. Selain itu, kursus ini mewajibkan setiap peserta kursus menginstall git dan memiliki akun github. Akun github tersebut dapat digunakan peserta kursus untuk mempublikasikan hasil pembuatan aplikasi android mereka.
- Sebelum memulai pengembangan aplikasi, peserta kursus perlu memahami Workspace dari Android Studio (IDE IntelliJ). Setelah itu, peserta kursus akan mempraktikkan deployment aplikasi android (dengan output yang bisa mereka lihat pada smartphone mereka) tanpa perlu memahami code.
